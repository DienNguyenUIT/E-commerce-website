## Run Locally

Open your terminal and run theses commands:
### 1. Clone repo

```
git clone https://github.com/DienNguyenUIT/SE104.L22.PMCL.git
cd SE104.L22.PMCL
```

### 2. Install NPM Packages
```
npm install
```
### 3. Run Application

```
npm start
```
